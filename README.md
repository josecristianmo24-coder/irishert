# Mi servidor de archivos por llave

Instrucciones rápidas:

1. Instala dependencias:

```bash
npm install
```

2. Arranca el servidor:

```bash
npm start
```

3. Abre en el navegador `http://localhost:3000` y usa el formulario para subir archivos.

Notas importantes:
- Este ejemplo guarda archivos en el disco local y usa una base de datos SQLite.
- Debes añadir moderación, escaneo antivirus y políticas de uso si piensas permitir contenido de terceros.
- No uses esto para distribuir contenido ilegal; el propietario del servidor es responsable.
